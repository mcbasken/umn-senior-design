Flip-Flops and Registers

Sequential circuits store state information using memory elements. The most basic is the flip-flop, which can store a single bit. Flip-flops are used to build larger structures like registers, counters, and state machines.

D Flip-Flop

The D flip-flop captures input D on the rising edge of the clock and stores it as output Q.

Registers

A register is a group of flip-flops (typically D flip-flops) used to store multi-bit values. A 4-bit register stores four bits in parallel, each controlled by the same clock signal.

  • Inputs: D0–D3
  • Outputs: Q0–Q3
  • Trigger: Shared clock signal
