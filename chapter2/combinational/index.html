Multiplexers and Demultiplexers

Multiplexers (MUX) and Demultiplexers (DEMUX) are foundational combinational devices used for data selection and routing.

Multiplexer (MUX)

A 2ⁿ-to-1 multiplexer has:

  • 2ⁿ input lines
  • n select lines
  • 1 output line

Example: 4-to-1 MUX

S1S0Y (Output)
00I0
01I1
10I2
11I3

Logic Expression:

Y = (¬S1 ∧ ¬S0 ∧ I0) ∨ (¬S1 ∧ S0 ∧ I1) ∨ (S1 ∧ ¬S0 ∧ I2) ∨ (S1 ∧ S0 ∧ I3)

Demultiplexer (DEMUX)

A 1-to-2ⁿ demultiplexer has:

  • 1 input line
  • n select lines
  • 2ⁿ output lines

Example: 1-to-4 DEMUX

S1S0Active Output
00Y0 = D
01Y1 = D
10Y2 = D
11Y3 = D
