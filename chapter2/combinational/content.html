Chapter 2: Multiplexers and Demultiplexers

Multiplexers (MUX) and Demultiplexers (DEMUX) are foundational combinational devices used for data selection and routing.

Multiplexer (MUX)

A 2ⁿ-to-1 multiplexer has:

  • 2ⁿ input lines
  • n select lines
  • 1 output line

Example: 4-to-1 MUX

S1S0Y (Output)
00I0
01I1
10I2
11I3

Logic Expression:

Y = (¬S1 ∧ ¬S0 ∧ I0) ∨ (¬S1 ∧ S0 ∧ I1) ∨ (S1 ∧ ¬S0 ∧ I2) ∨ (S1 ∧ S0 ∧ I3)

Demultiplexer (DEMUX)

A 1-to-2ⁿ demultiplexer has:

  • 1 input line
  • n select lines
  • 2ⁿ output lines

Example: 1-to-4 DEMUX

S1S0Active Output
00Y0 = D
01Y1 = D
10Y2 = D
11Y3 = D

Quiz: Check Your Understanding

1. How many select lines are needed for an 8-to-1 MUX?

  • A) 2
  • B) 3
  • C) 4
  • D) 8
Show Answer

The correct answer is: B) 3

Explanation: An 8-to-1 MUX requires 3 select lines since 2³ = 8.

2. What is the output of a DEMUX when no select lines are high?

  • A) All outputs are high
  • B) All outputs are low
  • C) The first output (Y0) is active
  • D) Undefined
Show Answer

The correct answer is: C) The first output (Y0) is active

Explanation: When select lines are 0, the DEMUX routes input to Y0.

3. True/False: A MUX routes a single input to one of many outputs.

  • A) True
  • B) False
Show Answer

The correct answer is: B) False

Explanation: A MUX selects one of many inputs and routes it to a single output. The description matches a DEMUX.
