Flip-Flops and Registers

Sequential circuits store state information using memory elements. The most basic is the flip-flop, which can store a single bit. Flip-flops are used to build larger structures like registers, counters, and state machines.

D Flip-Flop

The D flip-flop captures input D on the rising edge of the clock and stores it as output Q.

Registers

A register is a group of flip-flops (typically D flip-flops) used to store multi-bit values. A 4-bit register stores four bits in parallel, each controlled by the same clock signal.

  • Inputs: D0–D3
  • Outputs: Q0–Q3
  • Trigger: Shared clock signal

Quiz: Check Your Understanding

1. What is the output of a D flip-flop when D = 1 and a rising clock edge occurs?

  • A) 0
  • B) It toggles
  • C) 1
  • D) No change
Show Answer

The correct answer is: C) 1

Explanation: A D flip-flop captures the value of D on the rising edge of the clock. If D = 1 at that moment, the output becomes 1.

2. Which flip-flop toggles its output when both inputs are 1?

  • A) D flip-flop
  • B) SR flip-flop
  • C) JK flip-flop
  • D) T flip-flop
Show Answer

The correct answer is: C) JK flip-flop

Explanation: The JK flip-flop toggles its output when both J and K inputs are 1 on the clock edge.

3. How many D flip-flops are required to build an 8-bit register?

  • A) 4
  • B) 8
  • C) 16
  • D) 1
Show Answer

The correct answer is: B) 8

Explanation: One D flip-flop stores one bit. To store 8 bits, you need 8 D flip-flops.

4. True or False: Flip-flops are used only in combinational logic circuits.

  • A) True
  • B) False
Show Answer

The correct answer is: B) False

Explanation: Flip-flops are used in sequential logic circuits, where outputs depend on past inputs (i.e., state), not just current inputs.
